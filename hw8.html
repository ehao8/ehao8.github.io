REST API


DocuSign eSignature Rest API is an API that can be used to integrate valid eSignatures into any forms or applications. Instead of physically printing and signing forms or applications, then loading it back onto the computer, DocuSign eSignature API allows it all to be done electronically. It uses OAuth2 authentication to make calls to the API. It can return both JSON and XML response files depending on how the user sets up what response file will be sent back. Inside the response file is the data of the form or application that was filled. Each response file can be stored as a row entry in a database to be queried if needed. Documentation can be found here: https://developers.docusign.com/docs/esign-rest-api/
